 <center><h1><strong>Dynamic-Menu-for-laravel</strong></h1></center>
 
 <h3>Overview</h3>
Dynamic menu list that is  sortable using jquery-Sortable. 
This project is build with laravel and uses jquery library "sortable" for sorting the menu items.

<h3>Getting started:</h3>
<h5>Setting up database table</h5>
Make a database table with at least following fields.
<ul>
<li>id</li>
<li>Menu Name</li>
<li>parent_id</li>
<li>Order</li>
</ul>

<h5>Views,Routing,Controller and model are simple enough to understand.</h5>
<h5>Ordering of menu items and saving order is done with ajax request in "menuItemsort.js" inside public/js/menu directory.</h5>
<h5>Jquery needed are in public/js.</h5>
